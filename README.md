  A mysql database and application backend for an e-commerce site. Built using MySQL2, Express, Sequelize and dotenv.
  ## Installation
  npm init

  npm install mysql2 sequelize dotenv
  ## Usage
  source db/schema.sql

  quit

  npm run seed

  npm start
